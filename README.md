# 라이브마켓 셀러 사용법 — 영상 목록

셀러 어드민의 **사용법** 탭(`/{스토어}/admin/guide`)이 읽는 목록입니다.

**`videos.json` 하나만 고치면 화면이 바뀝니다. 배포도, 개발자도 필요 없습니다.**
GitHub에서 파일 열고 연필(✏️) 눌러 고친 뒤 저장(Commit changes) → 어드민 새로고침.
(반영까지 최대 5분. GitHub CDN 캐시 때문입니다.)

## 카테고리 추가

`sections` 목록에 이 블록을 통째로 하나 더 붙여넣고 내용만 바꾸세요.

```json
{
  "id": "live",
  "title": "라이브 방송",
  "description": "방송 켜기부터 채팅 주문까지.",
  "colorKey": "actionPrimary",
  "videos": []
}
```

`colorKey` (썸네일 색): `primaryMain`(빨강) · `dark`(검정) · `actionPrimary`(남색) · `gray1`(회색) · `infoDark`(파랑) · `successDarker`(초록)

## 영상 추가

해당 카테고리의 `videos` 목록에 붙여넣으세요.

```json
{
  "id": "order-chat",
  "title": "채팅으로 주문받기",
  "overline": "\"1번요\" 한마디로",
  "label": "주문챗",
  "why": "카드에 뜨는 설명 한 줄. 왜 봐야 하는지.",
  "duration": "6:44",
  "youtubeId": "여기에_유튜브_영상_ID",
  "keywords": ["주문", "채팅", "댓글"]
}
```

- **`youtubeId`**: 주소에서 따옵니다 — `youtube.com/watch?v=`**`dQw4w9WgXcQ`**
- **비워두면** 카드가 "촬영 예정"으로 뜹니다 (재생 안 됨)
- **`label`**: 썸네일에 크게 박히는 글씨 / **`overline`**: 그 위 작은 글씨
- **`keywords`**: 셀러가 질문창에 이 단어를 쓰면 이 영상을 추천합니다

## 순서 바꾸기

목록에서 블록 위치를 위아래로 옮기면 그 순서대로 뜹니다. 히어로("지금 할 것")도 이 순서를 따릅니다.

## "지금 하러 가기" 버튼 (선택)

영상에 `task`를 달면 해당 어드민 화면으로 가는 버튼이 붙습니다.

```json
"task": { "key": "timeDeal", "cta": "타임딜 만들러 가기", "doneLabel": "타임딜 만들어봄" }
```

`key`로 쓸 수 있는 값 (**앱 코드에 정의된 것만 됩니다 — 새 값을 쓰면 무시됩니다**):

| key | 어디로 가나 | 완료 자동 판정 |
|---|---|---|
| `timeDeal` | 타임딜 생성 | ✅ 타임딜 하나라도 만들면 |
| `product` | 상품 관리 | ✅ 상품 하나라도 있으면 |
| `productBulk` | 상품 관리 | ❌ 셀러가 "이미 했어요" |
| `productCopilot` | 상품등록 코파일럿 | ❌ 셀러가 "이미 했어요" |

## 챕터 (선택)

질문 답변이 **"2:36부터 보세요"** 로 꽂아줍니다. 유튜브 자막에서 뽑으면 편합니다.

```json
"chapters": [
  { "time": "1:10", "startSeconds": 70, "label": "무료배송 조건 거는 법" }
]
```

## 주의

- **JSON 문법이 깨지면 화면이 안 뜹니다.** 쉼표(`,`)와 따옴표(`"`)를 빠뜨리지 마세요. 저장 전에 [jsonlint.com](https://jsonlint.com)에 붙여넣어 검사하면 확실합니다.
- 마지막 항목 뒤에는 쉼표를 붙이지 않습니다.
